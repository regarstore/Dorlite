# How to use with TERMUX
1. Install git & python
```
pkg install python
pkg install git
```
2. Clone this repo
```
git clone https://github.com/flyxt/myxl-cli
cd myxl-cli
```
3. Install dependencies
```
pip install -r requirements.txt
```
4. Run the script
```
python main.py
```
